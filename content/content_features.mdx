### Highlighting Code

You can use the following syntax to highlight inside the code:
* highlight-line
* highlight-next-line
* highlight-start
* highlight-range{1, 4-6} (relative to the position in code)

**as a comment in your code!** It looks like this:

```js
	this.value = value; // highlight-line
	this.lender = lender;
	// highlight-next-line // highlight-next-line
	this.borrower = borrower;
	this.linearId = linearId;
	this.borrower = borrower;
	this.linearId = linearId;
	// highlight-range{2,4-6}
	this.borrower = borrower;
	this.linearId = linearId;
	this.borrower = borrower;
	this.linearId = linearId;
	this.borrower = borrower;
	this.linearId = linearId;
	this.borrower = borrower;
	this.linearId = linearId;
```

Where the first line in markdown actually contains this:

`this.value = value; // highlight-line`

See the [documentation](https://www.gatsbyjs.org/packages/gatsby-remark-prismjs/#line-highlighting) for further details.

## Latex

The premise is that if you have a public key $$(pubKey,n)$$ and a private key $$priKey$$ then you can:

* encrypt a message with the following:

$$message ^{pubKey} \equiv encMessage \mod n$$

* decrypt the encrypted message with the following:

$$encMessage ^{priKey} \equiv  message \mod n$$
